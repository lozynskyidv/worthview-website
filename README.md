# WorthView Website

Official website for WorthView - Everything you own and owe, in one place.

## Built with

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React (icons)

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Deployment

Configured for Netlify deployment. Push to GitHub and connect to Netlify for automatic deployments.

## Domain

worthview.app (configured in Namecheap, connected to Netlify)
